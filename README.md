# Studio Ghibli-Themed Cookie Clicker Game

## Overview

The **Studio Ghibli-Themed Cookie Clicker Game** is an engaging and visually appealing game where players click to earn cookies, inspired by the enchanting world of Studio Ghibli. Developed using Java and Android Studio, the game features whimsical animations and designs that bring the Studio Ghibli theme to life.

## Features

- **Cookie Clicking Mechanism:** Click on the cookie to earn points, with incremental rewards based on the number of clicks.
- **Studio Ghibli Theme:** Immerse yourself in a Studio Ghibli-inspired world with themed graphics and animations.
- **Animated Graphics:** Utilize `AnimationDrawable` to create smooth and charming animations that enhance the game experience.
- **Interactive UI:** Designed with XML layout files to ensure an intuitive and responsive user interface.
- **Upgrades and Bonuses:** Unlock various upgrades and bonuses to increase your cookie production rate.

## Usage

- **Launch the Game:**
  - Start the application on your Android device or emulator.

- **Play the Game:**
  - Click on the cookie to earn points. Watch the Studio Ghibli-themed animations and interact with various game elements.

- **Upgrade and Progress:**
  - Use the earned cookies to purchase upgrades and bonuses to boost your cookie production rate.

## Development Details

- **Programming Language:** Java
- **Game Development:** Developed using Android Studio with Java for game logic and functionality.
- **User Interface:** Designed with XML layout files to provide a responsive and user-friendly experience.
- **Animations:** Utilizes `AnimationDrawable` for creating smooth and delightful animations.

## Software Used

- **Java:** The primary programming language used to implement the game's logic and functionality.
- **XML:** Used for designing the user interface and layout of the game, ensuring a responsive and intuitive experience.
- **Android Studio:** The integrated development environment (IDE) used for building and managing the Android application.
- **AnimationDrawable:** Utilized for creating frame-by-frame animations to enhance visual appeal and user engagement.
